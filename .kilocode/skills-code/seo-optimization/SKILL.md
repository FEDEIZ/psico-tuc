# SEO Optimization

When building a landing page for a psychologist, ensure the following SEO practices are implemented:

## Meta Tags
- Include a unique `<title>` tag with the psychologist's name and services (e.g., "Dr. Jane Smith | Clinical Psychologist | City").
- Provide a `<meta name="description">` that summarizes the page content with relevant keywords (psychologist, therapy, mental health, etc.).
- Add `<meta name="keywords">` (optional but helpful for older search engines).
- Include viewport meta tag for responsiveness: `<meta name="viewport" content="width=device-width, initial-scale=1">`.

## Structured Data (Schema.org)
- Implement JSON‑LD markup for `LocalBusiness` and `Psychologist`.
- Include essential properties: `name`, `address`, `telephone`, `url`, `sameAs` (social media links), `openingHours`, `priceRange`, `serviceType`.
- Add `MedicalBusiness` and `MedicalSpecialty` where appropriate.

## Technical SEO
- Create a `robots.txt` file to guide search engine crawlers.
- Generate a `sitemap.xml` listing all important pages (even if single page).
- Ensure clean, semantic URLs (no hash‑based navigation for core content).
- Use canonical tags if necessary.

## On‑Page SEO
- Use heading tags (`h1`–`h6`) in a logical hierarchy; the main title should be an `h1`.
- Include target keywords naturally in headings and body text.
- Optimize images: descriptive filenames, `alt` attributes, and compression.
- Internal linking to other sections of the same page (anchor links) and external links to authoritative mental‑health resources.

## Performance & Mobile‑Friendliness
- Page load speed directly impacts SEO; follow performance optimization skill.
- Ensure the site is fully responsive and passes Google's Mobile‑Friendly Test.

## Monitoring
- Submit the sitemap to Google Search Console and Bing Webmaster Tools.
- Monitor rankings and crawl errors regularly.
